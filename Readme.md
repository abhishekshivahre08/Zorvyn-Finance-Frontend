# 🎨 Finance Dashboard - Frontend (React UI)

This is the client-side application for the Finance Dashboard, built with **React**, **Tailwind CSS**, and **Axios**. It connects to a secure Node.js/Express backend to provide a seamless financial management experience.

---

## 🔗 Project Links
- **🚀 Live Demo:** [https://finance-dashboard08.netlify.app/](https://finance-dashboard08.netlify.app/)
- **⚙️ Backend Repository:** [https://github.com/abhishekshivahre08/Zorvyn-Finance-Pulse.git]
- **📡 Live API:** [https://finances-dashboard.onrender.com/api](https://finances-dashboard.onrender.com/api)

---

## ✨ Features Implemented
- **User Auth:** Secure Login with JWT persistence.
- **Dashboard Summary:** Real-time visualization of Income, Expenses, and Balance.
- **Transaction Management:** View and manage financial records.
- **Role-Based UI:** UI elements adapt based on user roles (Admin/Viewer).
- **Responsive Design:** Mobile-friendly layout using Tailwind CSS.
